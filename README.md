# vue-boilerplate

## Project setup
```
npm install
```

### Compiles and hot-reloads for local development
```
npm run serve
```

### Compiles and minifies for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run prod
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
